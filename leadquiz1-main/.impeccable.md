## Design Context

### Users
North American university students — Sophomores through Recent Grads (Class 2024–2026) — scrolling LinkedIn, Reddit r/csMajors, Discord, or Instagram during recruiting season. They're anxious about behavioral interviews but won't engage with anything that feels like "prep." Core emotional hook: curiosity ("what do recruiters actually see?") and social comparison ("are my friends going to agree or expose me?").

### Brand Personality
Direct · Slightly uncomfortable · Genuinely useful

This is a viral lead quiz — a brand artifact distinct from the main MYLS product. The tone is Gen-Z casual: lowercase copy, conversational directness, a bit of friendly pressure. It should feel like a brutally honest friend, not a career coach.

Not: "We recommend improving your communication skills."
Yes: "you repeated yourself. say it once, then move forward."

### Aesthetic Direction
**Aura Scan** — dark, structured, AI-premium. Concentric rings + subtle grid texture convey the "aura scan" concept without gimmick.

**Palette: blue-only.** No pink/orchid (those belong to the main MYLS product, not this quiz).
- bg-base: #09091A
- bg-surface: #0F0F26
- bg-elevated: #161632
- primary: #3D51FF
- primary-light: #6D7CFF
- gradient: #BA8FFF → #717FFF (CTAs only)
- border-subtle: rgba(255,255,255,0.07)
- text hierarchy: #FFF / rgba(255,255,255,0.55) / rgba(255,255,255,0.30) / rgba(255,255,255,0.22)

**Typography:** DM Sans (Google Fonts CDN). 700–800 for headlines, 400–600 for body. No monospace, no secondary typeface.

**Anti-references:** glassmorphism, purple-glow AI slop, gradient text on headings, cyan accents, orchid/pink anything.

### Design Principles
1. **Coach tone, not tool tone** — copy should sound like a direct human ("you drifted"), never a system ("suboptimal structure detected")
2. **Blue-only brand** — gradient reserved for primary CTAs only; no pink, no secondary palette from main MYLS
3. **Delight through specificity** — micro-animations (radar draw, stagger reveals, unlock flash) should feel earned, not decorative
4. **Contrast drives conversion** — "this is still generic" → "we go way deeper" is the core unlock-page mechanism; maintain the before/after framing
5. **Mobile-first, single file** — 375px reference width, no build step, no server; all animations via CSS keyframes + minimal JS
